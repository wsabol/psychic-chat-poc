PHASE 2: EXTRACT ALL STRINGS (4 hours)
- Audit every .js/.jsx file
- Identify ALL hardcoded strings
- Create translation keys
- Add to JSON files

PHASE 3: UPDATE COMPONENTS (6-8 hours)
- Import useTranslation in every component
- Replace hardcoded strings with t() calls
- Update all templates and labels
- Handle dynamic text with variables

PHASE 4: TEST COMPREHENSIVE (2-3 hours)
- Test every page in all 8 languages
- Verify language persistence (localStorage + DB)
- Test user flows: Sign up → Chat → Billing → Settings
- Verify pre-auth and authenticated flows

PHASE 5: HANDLE ORACLE RESPONSES (2 hours)
- Determine if responses need backend translation
- OR handle response language formatting on client
- Test with actual chat responses

PHASE 6: FINAL QA (1-2 hours)
- Test all pages, all languages
- Verify database saves/loads
- Test language switching mid-flow
- Test offline functionality
